## ğŸ†˜ Troubleshooting

### Can't Access JARVIS

**Problem:** Site not loading

**Solutions:**
1. Wait 2-3 minutes after pushing changes
2. Clear browser cache (Ctrl/Cmd + Shift + R)
3. Try incognito/private mode
4. Check GitHub Pages is enabled in Settings

### Password Not Working

**Problem:** Password doesn't unlock

**Solutions:**
1. Verify you changed it in `script.js`
2. Check for typos
3. Wait for deployment (1-2 minutes)
4. Clear browser cache
5. Check browser console for errors

### Animations Laggy

**Problem:** UI is slow or choppy

**Solutions:**
1. Close other browser tabs
2. Update browser to latest version
3. Try different browser
4. Reduce number of particles in `script.js`

### Mobile Issues

**Problem:** Not working on mobile

**Solutions:**
1. Use Chrome or Safari (not in-app browsers)
2. Enable JavaScript
3. Clear mobile browser cache
4. Try landscape orientation
